TerryCharts

A framework for building Web visualization

基于canvas、svg的二维绘图引擎，目前处于研发阶段...

现支持三维贝塞尔曲线、圆、心形、线、矩形、玫瑰花形、扇形、n角星

examples:https://ytg123.github.io/TerryCharts/examples/index.html

合并分支测试

git merge 分支名

启动项目

```
npm run dev
```

打包

```
npm run build / npm run build:prod
```

文档启动

```
npm run storybook
```
